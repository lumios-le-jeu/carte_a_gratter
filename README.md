# Carte à Gratter Personnalisable ✨

Une application web moderne pour créer et partager des cartes à gratter virtuelles.

## Fonctionnalités
- **Créateur** : Choisissez une image/vidéo de fond, une couverture à gratter, et un message personnalisé.
- **Partage** : Générez un lien unique à envoyer à vos proches.
- **Support Multimédia** : Images et Vidéos supportées.
- **Design Premium** : Interface élégante "Glassmorphism" et animations fluides.

## Installation et Lancement

1. **Installation des dépendances**
   ```bash
   npm install
   ```

2. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

3. **Construction pour la production**
   ```bash
   npm run build
   ```

## Hébergement (GitHub Pages)

Pour héberger sur `https://github.com/lumios-le-jeu/carte_a_gratter` :

1. Créez le repository sur GitHub.
2. Poussez ce code sur la branche `main`.
3. Configurez GitHub Pages dans les paramètres du repo pour servir depuis `/dist` (nécessite une action de build) ou utilisez une branche `gh-pages`.

**Astuce** : Pour que les liens de partage fonctionnent, assurez-vous que l'URL de base est correcte. Si vous hébergez sur un sous-dossier (ex: `/carte_a_gratter/`), vous devrez peut-être ajuster le chemin de base dans `vite.config.js`.
